# SaberMayor

SaberMayor surge como respuesta a las dificultades que enfrentan los adultos mayores para acceder a información confiable sobre salud, tecnología y actividades de bienestar. Muchas veces, la falta de herramientas digitales adaptadas a sus necesidades limita su acceso a recursos que pueden contribuir a mejorar su calidad de vida y promover un envejecimiento activo.

## Descripción

SaberMayor es una plataforma orientada a brindar información y recursos de interés para adultos mayores, facilitando el acceso a contenidos relacionados con salud, bienestar, actividades recreativas y uso de herramientas tecnológicas. La propuesta busca promover la inclusión digital y contribuir al desarrollo de una vida más activa, saludable e independiente.

## Características

* Acceso a información sobre salud y bienestar.
* Recursos orientados al envejecimiento activo.
* Contenidos de apoyo para la inclusión digital.
* Interfaz accesible y amigable para adultos mayores.
* Navegación sencilla e intuitiva.
* Información organizada en una única plataforma.
* Diseño adaptado a las necesidades del público objetivo.
* Landing Page desarrollada para presentar la propuesta de valor.

## Tecnologías

* Frontend: HTML, CSS, JavaScript
* UI/UX: Figma (wireframes, mockups y prototipos)
* Diagramación y documentación: Lucidchart, Miro, Microsoft Word
* Gestión y trabajo colaborativo: GitHub, Trello, WhatsApp, Zoom
* Despliegue: GitHub Pages

## Proceso de Diseño

El proyecto fue desarrollado bajo un enfoque centrado en el usuario, utilizando:

* User Personas
* Empathy Maps
* User Journey Maps
* Lean UX Canvas
* Impact Mapping
* Product Backlog
* Wireflows
* Wireframes
* Mockups
* Prototipos de alta fidelidad

## Validación

Para asegurar que la solución responda a las necesidades de los usuarios, se realizaron:

* Entrevistas con usuarios.
* Validación de la Landing Page.
* Validación de prototipos.
* Pruebas de usabilidad.
* Evaluaciones heurísticas.
* Análisis de retroalimentación obtenida durante las validaciones.

Los resultados permitieron identificar oportunidades de mejora y validar la propuesta de valor de SaberMayor para los adultos mayores.
